# Status Codes
This section is for status codes in **Login** endpoint response.

|     Codes    |     Status    |
|---|---|
|     200    |     Success    |
|     401    |     Unauthorized    |
|     403    |     Forbidden    |
|     404    |     Not Found    |
|     405    |     Method Not Allowed    |
|     422    |     Unprocessable Entity    |
|     500    |     Internal Server Error    |
